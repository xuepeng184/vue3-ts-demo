# vue3+ts+element-plus 小项目

## 

## 基本描述

这个项目是使用了`vue3+ts+elemnt-plus`的一个小项目

==是仿b站项目，不是原创==

执行下面命令，下载项目所需依赖

```js
npm install
```

执行下面命令，启动服务

```
npm run serve
```
